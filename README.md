# Nevzat-Portfolio
# 🌌 Cyber Portfolio | Full-Stack Developer Showcase

Bu proje, **Nevzat** (Full-Stack Geliştirici) için özel olarak tasarlanmış, "Cyber Purple" (Siber Mor) konseptli, yüksek etkileşimli kişisel portfolyo ve ekip tanıtım sitesidir. 

Sıfır kütüphane bağımlılığı ile tamamen **Vanilla HTML, CSS ve JavaScript** kullanılarak geliştirilmiştir. Modern web standartlarına uygun, hızlı, duyarlı (responsive) ve Vercel üzerinden statik olarak yayınlanmaya hazır bir yapıdadır.

## ✨ Öne Çıkan Özellikler

* **Neon & Cyberpunk Estetiği:** Amethyst (Mor) ve Neon Pembe renk paletiyle oluşturulmuş derin karanlık tema.
* **Özel Animasyonlar:** CSS Keyframe'leri ile sıfırdan yazılmış "Glitch" (Bug) metin efektleri ve alt kısımda süzülen "Nebula" uzay bulutları.
* **Etkileşimli Kartlar:** Fare imlecini takip eden ve kartların etrafında parlayan özel "Glow" aydınlatma sistemi.
* **Terminal & Daktilo Efekti:** Karşılama ekranında çalışan interaktif terminal penceresi ve kendi kendini yazan (typing) metin algoritması.
* **Gizli Easter Egg:** Footer alanındaki yazıya tıklandığında açılan siber terminal modülü (`/hardware.info`).
* **Takım Ruhu:** Ekip üyelerinin (Nevzat, Bulut ve Alttre) yeteneklerini ve projeye katkılarını sergileyen özel "Takım Arkadaşları" bölümü.

## 🛠️ Kullanılan Teknolojiler

* **HTML5:** Semantik web yapısı.
* **CSS3:** Flexbox, CSS Grid, Custom Properties (Değişkenler) ve gelişmiş animasyonlar.
* **JavaScript (Vanilla):** DOM manipülasyonu, Intersection Observer API (scroll ile beliren elementler için) ve fare takibi.
* **Deployment:** [Vercel](https://vercel.com) (Statik dağıtım)

## 🚀 Kurulum & Çalıştırma

Bu proje statik bir web sitesi olduğu için herhangi bir sunucu kurulumuna veya paket yöneticisine (npm, pip vb.) ihtiyaç duymaz.

**Lokalde Çalıştırmak İçin:**
1. Projeyi klonlayın veya `.zip` olarak indirin.
2. Klasörün içindeki `index.html` dosyasını çift tıklayarak favori tarayıcınızda açın.

**Vercel'de Yayınlamak İçin:**
1. Bu repoyu kendi GitHub hesabınıza çatallayın (Fork/Push).
2. Vercel paneline girin ve "Add New Project" deyin.
3. Bu repoyu seçin ve hiçbir ayarı değiştirmeden doğrudan **Deploy** butonuna basın. (Flask veya Node.js ayarına gerek yoktur).
